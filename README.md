# Minesweeper

Versio miinaharavapelistä. Pelaaja paljastaa ruutuja, joiden alla on tyhjää, miina tai vinkki läheisistä miinoista. Tarkoitus on paljastaa kaikki ruudut, joiden alla ei ole miinaa.
